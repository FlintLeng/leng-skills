# 风雷翅 (Leng Fengleichi)

## 触发条件

当用户输入 **"风雷翅"** 后跟城市名称时，执行以下流程。

**示例：**
- "风雷翅 东京"
- "风雷翅 巴黎"
- "风雷翅 成都"

## 执行流程

### 1. 解析城市名称

从用户输入中提取城市名称：
- 输入格式：`风雷翅 <城市名称>`
- 支持中英文城市名称

### 2. 搜索旅游信息

使用多种信息源获取城市旅游攻略：

**信息源优先级：**
1. Web搜索（景点、美食、住宿）
2. 旅游网站（TripAdvisor、马蜂窝、携程等）
3. 本地生活指南

**搜索内容：**
- 必去景点 Top 10（含简介、地址、门票、开放时间）
- 必吃美食 Top 10（含推荐餐厅、人均消费）
- 推荐住宿位置（按区域分类、价格区间）
- 交通指南（机场/火车站到市区、市内交通）
- 最佳旅游时间
- 注意事项

### 3. 整理攻略格式

将信息整理为 Markdown 格式：

```markdown

|------|----------|------|------|------|----------|
| 1 | ... | ... | ... | ... | ... |

## 二、必吃美食 Top 10

| 排名 | 美食名称 | 推荐餐厅 | 人均消费 | 地址 |
|------|----------|----------|----------|------|
| 1 | ... | ... | ... | ... |

## 三、推荐住宿位置

### 1. <区域名称>
- **特点：** ...
- **价格区间：** ...
- **推荐酒店：** ...

## 四、交通指南

### 1. 如何到达
- **飞机：** ...
- **火车：** ...

### 2. 市内交通
- ...

## 五、最佳旅游时间

- ...

## 六、注意事项

- ...
```

### 4. 创建飞书文档

使用飞书 API 创建新文档：
- 文档标题：`<城市名称>旅游攻略 - YYYY年MM月`
- 使用 Python 脚本调用飞书 API

### 5. 写入文档内容

将整理好的攻略内容写入飞书文档。

### 6. 共享文档

为用户添加文档访问权限：
- 用户 open_id: `ou_85c6f41a2b841a35e049770984c555aa`
- 权限级别: `full_access`

## 脚本说明

### scripts/create_feishu_doc.py

创建飞书文档并写入内容的 Python 脚本（与搜魂术共用）。

**使用方法：**
```bash
python3 scripts/create_feishu_doc.py --title "文档标题" --content "Markdown内容"
```

## 错误处理

- 遇到访问受限、搜索失败、数据不可用等问题时，**跳过该数据源**
- 在执行详情中记录跳过原因
- 确保最终返回攻略结果，即使部分数据缺失

## 输出格式

### 执行详情

在攻略末尾附上执行详情：
- 各信息源获取状态（成功/跳过/失败）
- 跳过或失败的原因（如有）
- 文档链接
- 数据获取时间戳
