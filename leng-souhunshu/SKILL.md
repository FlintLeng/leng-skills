# 搜魂术 (Leng Souhunshu)

## 触发条件

当用户输入 **"搜魂术"** 时，执行以下流程。

## 执行流程

### 1. 获取全球重大新闻

使用多种新闻源获取近两日的全球重大新闻：

**新闻源优先级：**
1. CNBC (https://www.cnbc.com/world/)
2. NASDAQ API
3. 其他可访问的财经新闻网站

**新闻分类：**
- 政治 (Politics)
- 金融 (Finance)
- 经济 (Economy)
- 军事 (Military)
- 科技 (Technology)

**目标数量：** 每类约20条，总计100条

**遇到访问受限等问题时跳过，继续获取其他来源。**

### 2. 整理新闻格式

将新闻整理为 Markdown 格式，包含：
- 标题
- 分类
- 简要摘要（1-2句话）

### 3. 创建飞书文档

使用飞书 API 创建新文档：
- 文档标题：`全球重大新闻汇总 - YYYY年MM月DD-DD日`
- 使用 Python 脚本调用飞书 API

### 4. 写入文档内容

将整理好的新闻内容写入飞书文档：
- 使用飞书 docx API
- 支持标题、列表等格式

### 5. 共享文档

为用户添加文档访问权限：
- 用户 open_id: `ou_85c6f41a2b841a35e049770984c555aa`
- 权限级别: `full_access`

## 脚本说明

### scripts/create_feishu_doc.py

创建飞书文档并写入内容的 Python 脚本。

**使用方法：**
```bash
python3 scripts/create_feishu_doc.py --title "文档标题" --content "Markdown内容"
```

**环境变量：**
- `FEISHU_APP_ID`: 飞书应用 ID
- `FEISHU_APP_SECRET`: 飞书应用密钥
- 或直接在脚本中配置

## 错误处理

- 遇到访问受限、API超时、数据不可用等问题时，**跳过该数据源**
- 在执行详情中记录跳过原因
- 确保最终返回简报结果，即使部分数据缺失

## 输出格式

### 执行详情

在简报末尾附上执行详情：
- 各数据源获取状态（成功/跳过/失败）
- 跳过或失败的原因（如有）
- 文档链接
- 数据获取时间戳
